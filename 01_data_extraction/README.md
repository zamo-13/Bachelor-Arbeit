# DSA Extraction Provenance

## Extraction Tool

Name of extraction tool used: `dsa-tdb`

Purpose: used to download SoR records from the DSA Transparency Database public API.

What it downloaded: TikTok and X moderation records for the full year 2025 in Parquet format organized as daily chunks.

Output location: `C:\BA_Data\` (external to the repository due to storage constraints).

Note: the tool's source files have been removed from this repository as the extraction phase is complete. The tool is available at its original source for reference.
